# 时区地图工具

一个交互式的世界时区地图工具，可以查看世界各地的当地时间、时区和时差，并支持多城市时间对比。

## 功能特点

- 世界地图界面，支持点击任意位置查看时区信息
- 预设了60+个主要城市标记
- 显示当地时间（秒级实时更新）
- 显示时区信息
- 显示与本地时间的时差
- 支持添加多个城市到对比列表
- 支持搜索城市
- 深色主题设计

## 使用方法

### 方式一：直接打开

直接在浏览器中打开 `index.html` 文件即可使用。

### 方式二：使用本地服务器（推荐）

由于地图API可能需要HTTP/HTTPS环境，建议使用本地服务器：

```bash
# 使用 Python 3
python3 -m http.server 8000

# 或使用 Node.js (需要先安装 http-server)
npx http-server

# 然后在浏览器打开
# http://localhost:8000
```

## 交互说明

1. **点击地图** - 点击地图上任意位置查看该位置的时区信息
2. **点击城市标记** - 点击紫色圆点可快速定位到预设城市
3. **添加到对比列表** - 点击"+ 添加到对比列表"按钮将城市加入对比
4. **搜索城市** - 在搜索框输入城市名称（中文或英文）进行快速查找
5. **移除城市** - 悬停在已选城市上，点击"×"按钮移除
6. **清空全部** - 点击"清空全部"按钮移除所有已选城市

## 技术栈

- **Leaflet.js** - 地图显示
- **Luxon.js** - 时区处理
- **Axios** - API请求
- **CARTO Dark** - 深色地图瓦片
- **GeoNames API** - 时区数据

## 注意事项

- 时区API使用的是免费演示版，可能有请求限制
- 建议在网络良好环境下使用
- 某些区域可能无法精确获取时区信息
