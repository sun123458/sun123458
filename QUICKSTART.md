# AR增强现实展厅应用 - 快速启动指南

## 🚀 立即开始

### 步骤1: 启动应用

**方式A: 使用npm（推荐）**
```bash
npm install
npm start
```

**方式B: 使用Python**
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

**方式C: 使用已安装的http-server**
```bash
http-server -p 8000
```

### 步骤2: 打开应用

在浏览器中访问：**http://localhost:8000**

或直接运行：`npm run dev`（会自动打开浏览器）

### 步骤3: 准备Hiro标记

1. 打开 **hiro-marker.html** 文件
2. 点击"打印标记"按钮
3. 或直接将标记显示在另一个设备屏幕上

### 步骤4: 体验AR

1. 允许浏览器访问摄像头
2. 将摄像头对准Hiro标记
3. 等待3D展品出现
4. 开始交互体验！

- **交互模式**: 开启/关闭触摸操作

- **展品信息**: 查看展厅总体信息

## ⚠️ 常见问题

**Q: 无法识别标记？**
- 确保光线充足
- 调整摄像头到标记的距离（20-50cm）
- 打印效果要清晰

**Q: 3D模型不显示？**
- 检查摄像头权限是否允许
- 使用支持WebGL的浏览器
- 尝试刷新页面

**Q: 性能卡顿？**
- 减少同时显示的展品数量
- 在性能更好的设备上测试
- 确保设备有足够的内存
