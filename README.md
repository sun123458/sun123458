# 医疗管理界面系统

一个完整的诊所/医疗管理系统，支持患者管理、预约安排、用药提醒和生命体征记录。

## 功能特性

### 1. 患者管理
- 添加、编辑、删除患者信息
- 患者个人资料（姓名、年龄、联系方式、过敏史等）
- 头像上传支持
- 生命体征记录（血压、心率、体温、血糖）
- 高级搜索和筛选功能

### 2. 预约系统 (FullCalendar)
- 月/周/日视图切换
- 拖拽创建和调整预约
- 预约类型分类（常规检查、复查、急诊等）
- 冲突检测

### 3. 用药提醒
- 设置用药时间表
- 浏览器通知提醒
- 每日/自定义频率支持

### 4. 数据管理
- localStorage 本地存储
- 导出/导入 JSON 数据备份
- 完全离线工作

### 5. 无障碍设计
- 高对比度模式
- 可调节字体大小（正常/大/超大）
- 完整键盘导航支持
- ARIA 标签和屏幕阅读器支持
- WCAG 2.1 AA 标准

## 快速开始

### 方法 1: 直接在浏览器中打开

1. 打开 `index.html` 文件
   ```bash
   open index.html
   ```

### 方法 2: 使用本地服务器

由于浏览器安全限制，某些功能（如文件导入）建议使用本地服务器：

```bash
# Python 3
python -m http.server 8000

# Node.js (使用 npx)
npx serve

# PHP
php -S localhost:8000
```

然后访问: http://localhost:8000

## 使用指南

### 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Alt + 1` | 切换到仪表盘 |
| `Alt + 2` | 切换到患者库 |
| `Alt + 3` | 切换到预约日历 |
| `Alt + 4` | 切换到设置 |
| `Escape` | 关闭弹窗/对话框 |
| `Tab` | 在元素间导航 |

### 主要操作

1. **添加患者**
   - 点击 "患者库" → "+ 添加患者"
   - 填写基本信息、医疗信息和紧急联系人
   - 可选：上传患者头像

2. **创建预约**
   - 在 "预约" 视图中点击日期或时间段
   - 或点击 "+ 新建预约" 按钮
   - 选择患者并填写预约详情

3. **记录生命体征**
   - 打开患者详情 → "生命体征" 标签
   - 点击 "+ 添加记录"
   - 填写血压、心率、体温、血糖等数据

4. **设置用药提醒**
   - 打开患者详情 → "用药记录" 标签
   - 点击 "+ 添加提醒"
   - 设置药品名称、剂量和服药时间

5. **启用通知**
   - 前往 "设置" 视图
   - 点击 "请求通知权限"
   - 允许浏览器发送通知

## 数据备份

### 导出数据
1. 前往 "设置" 视图
2. 点击 "导出数据 (JSON)"
3. 数据文件将自动下载

### 导入数据
1. 前往 "设置" 视图
2. 点击 "导入数据"
3. 选择之前导出的 JSON 文件
4. 确认导入操作

## 注意事项

1. **数据存储**: 所有数据存储在浏览器的 localStorage 中，清除浏览器数据会导致数据丢失。请定期导出备份。

2. **通知权限**: 首次使用用药提醒功能时，需要授予浏览器通知权限。

3. **隐私**: 此系统完全在本地运行，不会将任何数据发送到外部服务器。

4. **头像**: 患者头像以 Base64 格式存储在 localStorage 中，请控制图片大小（建议 < 500KB）。
