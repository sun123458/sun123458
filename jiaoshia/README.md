# 虚拟教室平台原型 🎓

一个功能完整的虚拟教室平台原型，使用纯 HTML、CSS 和 JavaScript 构建，无外部依赖。

## 功能特性 ✨

### 1. 共享白板 🖌️
- **多种绘图工具**：画笔、橡皮擦、直线、矩形、圆形
- **自定义设置**：颜色选择器、可调节笔刷大小
- **Canvas 实现**：使用 HTML5 Canvas API 实现
- **触摸支持**：完全支持移动设备触摸绘图
- **清空功能**：一键清空白板

### 2. 参与者列表 👥
- 显示在线/离线状态
- 角色标识（主持人/学生）
- 头像和用户名显示
- 参与者计数

### 3. 聊天功能 💬
- 实时消息发送
- **表情符号支持**：12个常用表情快捷按钮
- 系统消息
- 消息时间戳
- 聊天记录本地存储
- 自动模拟其他用户回复

### 4. 屏幕共享模拟 📺
- 区域选择模式
- 可视化选择框（黄色虚线边框）
- 开始/停止共享状态
- 共享状态通知

### 5. 录制功能 ⏺
- 一键开始/停止录制
- 录制状态视觉指示（红点动画）
- 录制时长统计
- 录制通知

### 6. 响应式设计 📱
- 桌面端完整布局
- 平板适配（1024px以下）
- 移动端优化（768px以下）
- 触摸友好的交互
- 自适应画布大小

## 使用说明 📖

### 启动应用
1. 在浏览器中打开 `index.html` 文件
2. 或使用本地服务器：
   ```bash
   # 使用 Python
   python -m http.server 8000

   # 使用 Node.js
   npx serve
   ```

### 白板操作
- **选择工具**：点击工具栏中的工具图标
- **调整颜色**：点击颜色选择器选择颜色
- **调整笔刷**：拖动滑块调整笔刷大小（1-20px）
- **绘图**：在白色画布区域按住鼠标拖动
- **触摸绘图**：在移动设备上直接用手指绘图
- **清空画布**：点击垃圾桶图标

### 聊天操作
- **发送消息**：在输入框输入内容，点击"发送"或按回车
- **添加表情**：点击笑脸图标打开表情面板，选择表情
- **清空聊天**：点击聊天标题旁的垃圾桶图标

### 屏幕共享
1. 点击"屏幕共享"按钮
2. 在弹出的遮罩层中，用鼠标在白板上拖动选择区域
3. 点击"确认共享"开始共享
4. 共享时会有高亮边框显示
5. 再次点击按钮可停止共享

### 录制
- 点击"录制"按钮开始录制
- 录制时右上角会显示红色闪烁指示器
- 再次点击停止录制，系统会显示录制时长

## 技术实现 🔧

### 核心技术
- **HTML5 Canvas API**：白板绘图
- **Touch Events API**：移动设备触摸支持
- **Local Storage API**：聊天记录持久化
- **CSS Flexbox**：响应式布局
- **CSS Grid**：工具栏布局
- **Vanilla JavaScript**：无框架依赖

### 浏览器兼容性
- Chrome/Edge 90+
- Firefox 88+
- Safari 14+
- 移动浏览器（iOS Safari, Chrome Mobile）

### 响应式断点
- 桌面：> 1024px
- 平板：768px - 1024px
- 手机：< 768px
