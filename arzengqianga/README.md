# AR增强现实展厅应用

基于Web技术栈（Three.js + AR.js + HTML5）开发的AR增强现实展厅应用。

## 功能特性

- **AR识别**: 使用AR.js实现增强现实标记识别
- **3D渲染**: 基于Three.js的高质量3D模型渲染
- **交互控制**: 支持鼠标和触摸交互，可以旋转、缩放、平移展品
- **响应式设计**: 自适应各种屏幕尺寸，支持移动设备
- **用户界面**: 现代化的UI设计，包含控制面板、信息展示等
- **性能优化**: 实时FPS监控和性能优化

## 技术栈

- **Three.js**: 3D图形渲染库
- **AR.js**: 增强现实库
- **HTML5**: 结构和语义化标记
- **CSS3**: 样式和动画
- **JavaScript (ES6+)**: 应用逻辑

## 安装与运行

### 1. 克隆或下载项目

```bash
cd AR增强现实展厅应用
```

### 2. 安装依赖（可选）

```bash
npm install
```

### 3. 启动本地服务器

由于浏览器安全策略，AR功能需要在Web服务器环境下运行：

```bash
# 使用npm脚本
npm start

# 或使用其他服务器
npx http-server -p 8080
```

### 4. 访问应用

在浏览器中打开：`http://localhost:8080/html/index.html`

## AR标记使用

### 下载AR标记

1. 打开应用后，点击右下角的"帮助"按钮
2. 在帮助对话框中点击"下载AR标记"按钮
3. 将下载的标记图片打印出来

### 使用AR标记

1. 允许浏览器访问摄像头
2. 将摄像头对准打印的AR标记
3. 应用会自动识别标记并在其上方显示3D展品

## 功能说明

### 控制面板

- **重置视角**: 恢复所有展品到初始状态
- **自动旋转**: 开启/关闭展品自动旋转
- **显示网格**: 显示/隐藏场景网格辅助线
- **显示标签**: 显示/隐藏展品标签

### 交互操作

- **点击展品**: 查看展品详细信息
- **拖拽场景**: 旋转选中的展品
- **滚轮/双指**: 缩放展品
- **Shift + 拖拽**: 平移展品
- **R键**: 重置视角
- **G键**: 切换网格显示
- **ESC键**: 取消选择

### 展品类型

1. **雕塑**: 现代艺术风格的抽象雕塑
2. **花瓶**: 古代瓷器花瓶
3. **科技**: 未来科技概念模型

## 配置说明

配置文件位于 `js/config.js`，主要包括：

- **AR配置**: 标记类型、尺寸、标记文件路径
- **场景配置**: 背景色、光照、雾效等
- **展品配置**: 展品的类型、位置、颜色等
- **交互配置**: 旋转、缩放、平移的灵敏度
- **UI配置**: 语言、调试模式等

## 自定义开发

### 添加新展品

在 `js/config.js` 的 `exhibits` 数组中添加新的展品配置：

```javascript
{
    id: 'exhibit-4',
    name: '新展品名称',
    description: '展品描述',
    position: { x: 0, y: 0.5, z: 0 },
    rotation: { x: 0, y: 0, z: 0 },
    scale: { x: 1, y: 1, z: 1 },
    color: 0xff6b6b,
    type: 'sculpture',
}
```

### 创建自定义模型

在 `js/models.js` 中添加新的模型创建方法：

```javascript
createCustomModel(config) {
    // 创建你的3D模型
    const geometry = new THREE.YourGeometry();
    const material = new THREE.MeshPhongMaterial({
        color: config.color,
    });
    const mesh = new THREE.Mesh(geometry, material);
    return mesh;
}
```

## 故障排除

### 摄像头无法访问

1. 确保已允许浏览器访问摄像头
2. 检查是否有其他应用正在使用摄像头
3. 尝试使用HTTPS环境（摄像头权限要求）

### AR标记无法识别

1. 确保打印的标记清晰完整
2. 保证良好的光照条件
3. 保持摄像头与标记的适当距离
4. 避免标记表面反光

### 性能问题

1. 关闭其他占用资源的浏览器标签
2. 降低展品数量
3. 在配置文件中设置 `lowQualityMode: true`
