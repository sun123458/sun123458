# 虚拟教室平台原型

一个功能完整的虚拟教室平台原型，使用 HTML5 Canvas API 实现共享白板功能。

## 功能特性

### 1. 共享白板
- 使用 HTML5 Canvas API 实现
- 支持画笔和橡皮擦工具
- 可调节画笔颜色和大小
- 支持撤销操作
- 支持清空白板
- 支持触摸绘图（移动设备）
- 响应式设计，自动适应屏幕大小

### 2. 参与者管理
- 显示在线参与者列表
- 显示参与者状态（在线/主持人）
- 显示麦克风状态图标
- 说话时高亮显示参与者
- 参与者数量统计

### 3. 实时聊天
- 支持发送文字消息
- 内置表情选择器
- 区分自己和他人的消息
- 显示消息发送时间
- 自动滚动到最新消息
- 模拟自动回复功能

### 4. 屏幕共享模拟
- 点击选择共享区域
- 可选择共享白板或屏幕区域
- 共享时显示高亮边框动画
- 共享状态指示

### 5. 录制功能
- 一键开始/停止录制
- 录制状态视觉指示器（红色闪烁点）
- 录制按钮状态切换

### 6. 响应式设计
- 桌面端：三栏布局（参与者列表 | 主内容区 | 聊天）
- 移动端：单栏布局，侧边栏可折叠
- 触摸设备优化
- 自适应白板大小

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 无需安装任何依赖或服务器

## 技术栈

- 纯 HTML5、CSS3 和 JavaScript (ES6+)
- HTML5 Canvas API
- Flexbox 和 CSS Grid 布局
- CSS 动画和过渡效果
- Touch Events API（触摸支持）
- 无第三方依赖

## 主要技术实现

### 白板绘图
- 监听鼠标和触摸事件
- 使用 Canvas 2D Context 绘制路径
- 支持高DPI屏幕优化
- 路径历史记录用于撤销功能

### 状态管理
- 使用类来管理应用状态
- 本地存储白板绘图历史
- 本地管理聊天消息和参与者信息

### 响应式设计
- 媒体查询适配不同屏幕尺寸
- 移动端侧边栏折叠/展开
- 触摸事件处理
