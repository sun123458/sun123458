# Mingle - 在线配对交友卡片应用

一个功能完整、视觉精美的配对交友卡片应用，支持流畅的手势交互。

## 功能特性

### ✨ 核心功能
- **左右滑动卡片** - 向右滑动表示喜欢，向左滑动表示跳过
- **向上超喜欢** - 向上快速滑动发送超喜欢信号
- **卡片回弹动画** - 滑动未达到阈值时卡片平滑回弹
- **查看用户简介** - 点击信息按钮查看详细资料和更多照片
- **匹配成功弹窗** - 随机触发匹配成功动画和庆祝效果

### 🎨 视觉设计
- 年轻化渐变配色（粉色 + 紫色主题）
- 圆角卡片设计
- 流畅的CSS3动画过渡
- 在线状态、新用户标签
- 认证用户标识

### 📱 交互体验
- 支持鼠标拖拽（桌面端）
- 支持触摸滑动（移动端）
- 底部操作按钮快捷操作
- 滑动指示器实时反馈
- 匹配成功庆祝纸屑效果

## 使用方法

### 快速启动

1. 直接用浏览器打开 `index.html` 文件
2. 或使用本地服务器：

```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx serve
```

### 操作说明

| 操作 | 效果 |
|------|------|
| 向右滑动/点击 ❤️ | 喜欢该用户 |
| 向左滑动/点击 ✖️ | 跳过该用户 |
| 向上快速滑动/点击 ⭐ | 发送超喜欢 |
| 点击 ℹ️ 按钮 | 查看用户详细资料 |
| 滑动不够远 | 卡片回弹原位 |

## 技术栈

- **HTML5** - 语义化结构
- **CSS3** - 动画、渐变、响应式设计
- **原生 JavaScript** - 无依赖，轻量级
- **贝塞尔曲线** - 流畅的动画过渡

## 自定义数据

在 `script.js` 中可以自定义用户数据：

```javascript
const users = [
    {
        id: 1,
        name: '用户名',
        age: 23,
        location: '城市',
        avatar: '头像URL',
        photos: ['照片URL1', '照片URL2', '照片URL3'],
        bio: '个人简介',
        tags: ['标签1', '标签2'],
        online: true,
        isNew: true,
        verified: true
    }
];
```
