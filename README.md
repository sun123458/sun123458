# 医疗管理系统

一个功能完善的医疗管理界面，专为诊所和医疗机构设计，采用现代化的 Web 技术构建。

## 功能特性

### 1. 仪表板
- 患者总数统计
- 今日预约统计
- 活跃用药提醒统计
- 直观的数据概览

### 2. 患者管理
- ✅ 完整的患者个人资料页面
- ✅ 患者信息的创建、编辑和删除
- ✅ 高级搜索功能（支持姓名、ID、诊断搜索）
- ✅ 多条件筛选（年龄、性别、诊断）
- ✅ 患者详细资料查看
- ✅ 医疗备注和过敏史记录
- ✅ LocalStorage 数据持久化

### 3. 预约管理系统
- ✅ 使用 FullCalendar 实现的可视化日历
- ✅ 创建、编辑和删除预约
- ✅ 支持月、周、日视图切换
- ✅ 预约类型分类（咨询、体检、复查、急诊）
- ✅ 拖拽调整预约时间
- ✅ 颜色编码区分预约类型
- ✅ 患者关联预约

### 4. 用药提醒系统
- ✅ 浏览器通知 API 集成
- ✅ 创建个性化用药提醒
- ✅ 设置提醒时间、频率和剂量
- ✅ 声音提示功能
- ✅ 激活/停用提醒开关
- ✅ 提醒历史跟踪
- ✅ 支持一次性、每日、每周频率

### 5. 生命体征图表
- ✅ 使用 Chart.js 实现数据可视化
- ✅ 血压趋势图表（收缩压/舒张压）
- ✅ 心率变化图表
- ✅ 动态数据更新
- ✅ 交互式图表提示
- ✅ 高对比度模式适配

### 6. 高级搜索和筛选
- ✅ 实时搜索过滤
- ✅ 多维筛选组合
- ✅ 年龄分组筛选
- ✅ 性别筛选
- ✅ 诊断类型筛选
- ✅ 一键清除筛选

### 7. 无障碍功能（Accessibility）
- ✅ 高对比度模式
- ✅ 可调节字体大小（12px - 24px）
- ✅ 键盘导航支持
- ✅ 屏幕阅读器兼容
- ✅ ARIA 标签完善
- ✅ 焦点陷阱（模态框）
- ✅ 跳转到主内容链接
- ✅ 键盘快捷键：
  - `Ctrl/Cmd + C`: 切换高对比度
  - `Ctrl/Cmd + Plus`: 增大字体
  - `Ctrl/Cmd + Minus`: 减小字体
  - `Ctrl/Cmd + 0`: 重置无障碍设置

## 技术栈

- **HTML5**: 语义化标签，无障碍支持
- **CSS3**: 现代化布局，CSS 变量，响应式设计
- **JavaScript (ES6+)**: 模块化开发，LocalStorage API
- **FullCalendar 6.1**: 功能丰富的日历库
- **Chart.js 4.4**: 数据可视化图表库
- **Font Awesome 6.5**: 图标库

## 数据存储

所有数据都存储在浏览器的 LocalStorage 中，确保：
- 数据持久化
- 客户端隐私保护
- 快速数据访问
- 无需服务器依赖

**存储键**:
- `medical_patients`: 患者数据
- `medical_appointments`: 预约数据
- `medical_reminders`: 用药提醒数据
- `notification_permission`: 通知权限状态

## 使用说明

### 快速开始

1. 直接在浏览器中打开 `index.html` 文件
2. 系统会自动加载示例数据
3. 开始添加患者、创建预约和设置提醒

### 启用通知

1. 点击"用药提醒"标签
2. 点击"启用通知"按钮
3. 浏览器会请求通知权限，点击"允许"
4. 提醒将在设定时间自动弹出

### 高对比度模式

点击顶部无障碍工具栏的"高对比度"按钮，或使用快捷键 `Ctrl/Cmd + C`

### 调整字体大小

- 点击"+"按钮增大字体
- 点击"-"按钮减小字体
- 或使用快捷键 `Ctrl/Cmd + Plus/Minus`

- 演示和测试
- 个人使用
- 小规模诊所试点

对于生产环境，建议：
1. 实现服务器端 API
2. 添加用户认证
3. 数据加密
4. 定期备份

## 开发建议

### 添加新功能
- 所有模块都是独立的
- 易于扩展和维护
- 遵循现有代码风格

### 自定义样式
- 所有颜色使用 CSS 变量
- 修改 `:root` 中的变量即可

### 添加新的患者字段
1. 在 `patients.js` 中更新表单
2. 在 `samplePatients` 中添加示例数据
3. 在 `viewPatientDetails` 中渲染新字段
